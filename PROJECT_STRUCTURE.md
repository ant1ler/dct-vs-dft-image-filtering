## Папка `src/` — исходный код

- **`src/__init__.py`**  
  Чтобы `src/` импортировался как пакет.

### 1. Преобразования: `src/transforms/`

Ручные реализации преобразований (без `numpy.fft`/`scipy.fft`/`cv2.dct`).

- **`src/transforms/__init__.py`**  
  Пакетная инициализация (можно позже использовать для удобных импортов).

- **`src/transforms/dft_manual.py`**  
  DFT/FFT (ручная реализация):
  - `dft_1d`, `idft_1d` — прямое и обратное DFT по определению;
  - `fft_1d`, `ifft_1d` — ручная реализация FFT (Кули–Тьюки) и обратного FFT;
  - `dft_2d`, `idft_2d` — 2D DFT/IDFT (по строкам и столбцам);
  - `fft_2d`, `ifft_2d` — 2D FFT/IFFT.

- **`src/transforms/dct_manual.py`**  
  DCT (ручная реализация):
  - `_dct_matrix` — построение ортонормированной матрицы DCT-II;
  - `dct_1d`, `idct_1d` — 1D DCT-II и обратное DCT (DCT-III);
  - `dct_2d`, `idct_2d` — 2D DCT/IDCT (построчно + по столбцам).

### 2. Фильтры и шум: `src/filters/`

Тут фильтры (маски) и позже будут модели шума.

- **`src/filters/__init__.py`**  
  Пакетная инициализация.

- **`src/filters/frequency_filters.py`**  
  Маски частотных фильтров:
  - `ideal_low_pass(shape, cutoff)` — идеальный НЧ-фильтр (круглая маска);
  - `ideal_high_pass(shape, cutoff)` — идеальный ВЧ-фильтр (дополнение к НЧ);
  - `gaussian_low_pass(shape, sigma)` — гауссов НЧ-фильтр;
  - `gaussian_high_pass(shape, sigma)` — гауссов ВЧ-фильтр.

- **`src/filters/noise_models.py`**
  Модели синтетического шума:
  - `add_gaussian_noise(...)`;
  - `add_salt_pepper_noise(...)`.

### 3. Метрики качества: `src/metrics/`

Метрики качества для сравнения результатов.

- **`src/metrics/__init__.py`**
  Пакетная инициализация.

- **`src/metrics/quality_metrics.py`**  
  - `mse(original, restored)` — среднеквадратичная ошибка;
  - `psnr(original, restored)` — пик-сигнал-шум;
  - `ssim_rgb(original, restored)` — SSIM как единая RGB-метрика;
  - `ssim_channels_mean(original, restored)` — средний SSIM по каналам.

### 4. Эксперименты: `src/experiments/`

Скрипты, которые запускают эксперименты.

- **`src/experiments/__init__.py`**
  Пакетная инициализация.

- **`src/experiments/compare_dct_dft.py`**  
  Основной сценарий сравнения:
  - режим `paired` (готовые пары `raw/noisy`);
  - режим `synthetic` (шум генерируется в коде);
  - фильтрация через DFT/FFT и DCT (по каналам RGB);
  - расчёт MSE/PSNR/SSIM;
  - сохранение CSV и изображений результатов.

### 5. Визуализация: `src/visualization.py`

Вспомогательные функции для картинок/графиков:

- отображение исходного/зашумлённого/отфильтрованного изображения;
- отображение амплитудного/фазового спектра;
- построение графиков зависимости метрик (PSNR, MSE) от параметров фильтра.

## Папка `data/` — данные

Изображения и промежуточные результаты.

- **`data/raw/`**  
  Исходные RGB изображения (оригиналы, без шума).

- **`data/noisy/`**  
  Те же RGB изображения, но с добавленным шумом (paired-сценарий).

- **`data/results/`**  
  Результаты после фильтрации (например, подпапки `dct/`, `dft/` при необходимости).

Рекомендуемая схема набора данных для диплома:

- всего 60–80 clean изображений;
- около 60% для `paired`;
- около 40% для `synthetic`.
